AI Chatbot Widget Embeds

Revolutionary chatbot technology that embeds interactive widgets directly in conversations, creating seamless user experiences without page navigation.

Common Ninja's AI Chatbot will support dynamic embedding of widgets within the conversation flow. This innovative feature allows the chatbot to insert functional widgets in response to user intent, enabling direct interaction without requiring navigation to other parts of the site.
1

📅 Booking Widget

Trigger example: User asks to schedule a session, appointment, or meeting.
Chatbot behavior: Displays a booking interface that includes duration, location, and calendar navigation for date selection.
2

⏰ Countdown Widget

Trigger example: User asks about the launch date or time remaining for an event or product release.
Chatbot behavior: Embeds a countdown showing remaining time (days, hours, minutes) until the event.
3

⭐ Google Reviews Widget

Trigger example: User asks for customer reviews, ratings, or testimonials.
Chatbot behavior: Displays Google Reviews in carousel or card format with reviewer name, rating, and text.
4

🎟️ Coupon Code Widget

Trigger example: User asks for discounts, promotions, or coupon codes.
Chatbot behavior: Embeds a coupon code with a "Shop Now" button, including discount messaging and visual styling.
5

🎰 Scratch Card Widget

Trigger example: User asks about special offers or interactive promotions.
Chatbot behavior: Displays a virtual scratch card that reveals a randomized discount amount upon interaction.

Additional Features

🎯

Smart Intent Detection

Widget rendering triggered by detected user intent using keyword patterns and LLM-driven classification.

📱

Responsive Design

Embeds maintain responsive layout and full functionality within both desktop and mobile chat environments.

🔧

Platform Integration

Widgets are part of Common Ninja's existing platform and can be embedded contextually in the chat interface.

🚀

Expandable System

Initial launch supports five widget types, with expansion planned based on user demand and widget capabilities.
